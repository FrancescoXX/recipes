# Rust Hello World - Local

<!-- #ZEROPS_EXTRACT_START:intro# -->
This is a local development environment for [Rust Hello World (info + deploy)](https://app.zerops.io/recipes/rust-hello-world?environment=local) recipe on [Zerops](https://zerops.io).

A single PostgreSQL database running in the cloud — the developer runs the Rust application locally and connects to it using `zcli vpn up`, which bridges the local machine into the Zerops project network.
