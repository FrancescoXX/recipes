# Rust Hello World - Local
This is a local development environment for [Rust Hello World (info + deploy)](https://app.zerops.io/recipes/rust-hello-world?environment=local) recipe on [Zerops](https://zerops.io).

A production-mode `app` service is deployed to the cloud alongside a PostgreSQL database. The Rust application runs locally on your machine — use `zcli vpn up` to connect your local environment directly to the Zerops private network and reach the database at its internal hostname.
