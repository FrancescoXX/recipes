# Rust Hello World - AI Agent

<!-- #ZEROPS_EXTRACT_START:intro# -->
This is an AI agent development environment for [Rust Hello World (info + deploy)](https://app.zerops.io/recipes/rust-hello-world?environment=ai-agent) recipe on [Zerops](https://zerops.io).

Two application containers and a PostgreSQL database: `appdev` (dev setup) provides a full Rust toolchain workspace for the agent to compile and run the app via SSH, while `appstage` (prod setup) allows testing the complete release build pipeline end to end.
