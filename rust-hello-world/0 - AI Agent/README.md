# Rust Hello World - AI Agent
This is AI agent environment for [Rust Hello World (info + deploy)](https://app.zerops.io/recipes/rust-hello-world?environment=ai-agent) recipe on [Zerops](https://zerops.io).

<!-- #ZEROPS_EXTRACT_START:intro# -->
Actix Web application connected to PostgreSQL, with a development container (appdev) for agent-driven source editing and iterative `cargo run` sessions via SSH, and a staging container (appstage) for validating production builds through the full `cargo build --release` pipeline.
