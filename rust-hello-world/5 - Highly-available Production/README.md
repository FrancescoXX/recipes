# Rust Hello World - Highly-available Production
This is a highly-available production environment for [Rust Hello World (info + deploy)](https://app.zerops.io/recipes/rust-hello-world?environment=highly-available-production) recipe on [Zerops](https://zerops.io).

An `app` service runs on dedicated CPU cores with a minimum of two containers, connected to a HA-mode PostgreSQL database with automatic primary/replica failover — all on a Serious core project for isolated, enterprise-grade infrastructure.
